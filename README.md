Saving you from having to click multiple times to copy the BibTex of a paper on Google Scholar.

Existing interaction design on Google Scholar requires you to click multiple items and navigate to a new page to access the BibTex code.

Our extension adds a button that allows one-click copying of BibTex code of a paper.

Available from the [Chrome store](https://hci.ucla.edu/bib).
